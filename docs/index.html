Get Started in Minutes

Simple installation, powerful features

1

Clone & Install

git clone https://github.com/Dorky-Robot/katulong.git cd katulong npm install
2

Start the Server

npm start

Server starts on ports 3001 (HTTP) and 3002 (HTTPS)

3

Register Your Device

Open http://localhost:3001 in your browser. Follow the WebAuthn registration flow to create your first passkey.

4

Pair Additional Devices

From Settings â†’ "Pair New Device", scan the QR code on your mobile device and enter the PIN to pair securely.
